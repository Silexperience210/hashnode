Identity & Lightning Wallet

Connect your Nostr identity (you'll be the admin of this node) and link your Lightning wallet via Nostr Wallet Connect (NWC).

Your Nostr public key — grants you admin access. Install Alby or nos2x to auto-fill.

Find your NWC string in Umbrel → Alby Hub, Start9, or Mutiny Wallet.

Leave blank if you only want local network access. For internet access, the installer sets up a Cloudflare Tunnel automatically — run sudo journalctl -u hashnode-tunnel -f to get the URL and paste it here. This URL is broadcast to other HashNode nodes via Nostr so they can link to your marketplace.
